* `db.collection.dataSize()`


| <br /> |                                                       |
| ------ | ----------------------------------------------------- |
| 返回： | 集合的大小(以字节为单位)。 数据压缩不会影响此 value。 |


此方法在collStats(即：db.collection.stats())命令的尺寸输出周围提供 wrapper。
